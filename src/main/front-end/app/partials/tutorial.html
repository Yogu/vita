The Preview

1 The Import Panel – the user can set the desired analysis parameters

2 The list of imported documents, correctly imported documents are marked with a checkmark, failed documents with an exclamation mark and still non-completed with the play icon

3 A short preview of the imported/chosen document

4 The user can set the analysis parameters to reanalyze the document

ViTA allows the import of text-, epub2- and epub3-documents. Before the import the user has the possibility before the import to set the analysis parameters. The user can set the count for the Word Cloud items and the granularity in the Graph Network. He can enable the stop word list and the filter for unlikely person and place names. At last he can choose between three NLP tools: ANNIE, OpenNLP and Stanford.
The Preview also provides the user with a list of all documents which he had imported. If there is a checkmark before the title of the document in the list, the analysis was correctly executed. If there is an exclamation mark before the title, the analysis has failed. The play icon before the title illustrates the procedure of the analysis. The user has the ability to cancel the analysis of processing documents by clicking on the cancel icon next to the trash icon. He also has the ability to restart the analysis of the canceled documents, by clicking in the restart icon next to the trash icon.
On the right, there is a preview of the chosen document displayed. Again the user has the possibility to restart the analysis, but with new analysis parameters by clicking on Analysis Parameters next to the Rename button below the preview. The new analyzed document will be add to the list.

The Overview

1 The list of offered interaction possibilities, including the Fingerprint, the Graph Network, the Word Cloud, the Plot View and the profile of a character/place

2 The general information about the document: the name of the author, the publish year, the main characters, the number of chapters and the total count of words

3 The corresponding pictures to the list of offered interaction

The Overview – a document is imported/chosen and the analysis and interaction can start. This view gives the user some general facts about the document, for example the most frequent occurred characters. Furthermore the Overview gives you links to the various analyzations. Thereby, the user has the opportunity to choose the right label in the list of interaction possibilities on the left, or he can select the corresponding picture. Besides the Overview, the tool ViTA offers 7 other views on the chosen document.

The Graph Network

1 Intersection Fingerprint between two characters parallel to the edge between the two nodes

2 The visualization Graph Network consists of nodes and edges. Nodes represents characters, edges represents the connection between two characters. At the beginning the most important characters are already selected

3 Reset - deselect all added characters, the most important characters are still selected
Deselect all - deselect all characters

4 Search field – the user can search a character to select him for the Graph Network

4 The user can set the range to get the Graph Network for a certain interval

The visualization Graph Network is a representation with nodes and edges of the connection between the characters in the document. Each node represents a character and each edge represents the connection between the two characters. The more important the character, the bigger the node. The more the characters occurs together in a sentence, the wider the edge between the nodes. Right at the start, the most important characters (with the maintained color) are already selected and shown in the visualization. The most common characters with the assigned color are shown in the Graph Network with the same color. The user can add more characters to his selection or deselect characters by clicking on the desired name in the list on the right. With the selection, the border of the name is getting fat. If the user clicks on an edge between two nodes, the Fingerprint between the two characters will be displayed above the Graph Network. With a click on the citation in the Fingerprint, the user is forwarded to the Document. The Graph Network includes an important feature. The user has the ability to set the range for the Graph Network, for example he just wants to get the Graph Network of the interval between chapter 3 and chapter 6.

The Word Cloud

1 The visualization Word Cloud – the common the word occurs, the bigger is the size of the word

2 Search field – the user can search a character to select him for the Word Cloud

3 The list of characters, the most common characters are on the top and have their own color

The visualization Word Cloud is a known visualization. The Word Cloud has the functionality, to get a quick gathering of the content. Common words in the document are displayed in a bigger size than less common. In this case, the user has the opportunity to get the Word Cloud Word Cloud of the whole document with a click on global in the list on the right. Otherwise he can get the Word Cloud of a single character. This Word Cloud is compounded of words, which often occurs with the chosen character in a sentence. The colors of the most common characters are maintained and the characters are displayed with the assigned color in the Word Cloud.

The Plotview

The Plot View is a specific visualization. The 10 most significant characters are illustrated as lines. Each character line has an own color. This color has no colligation to the color which is assigned to the character in the other visualizations. The lines represents the history of the document. If a characters name is occurred afterwards in the Plot View, the character appears later in the history of the document. Each node represents a chapter of the document, except the first nodes next to the names. The first nodes only illustrates the start of the occurrence of the characters. The Tooltip over every node displays the title of the chapter and lists the occurred characters.

The Fingerprint

1 The Combination Fingerprint – Fingerprint of multiple characters

2 With a click on Toggle seperators the user can set the parts and chapters into the Fingerprint

The Fingerprint is a known visualization too. Each character has a specific bar. The bar symbolizes the whole document. The start of the bar represents the beginning and the end of the bar the ending of the document. The Fingerprint illustrates the occurrence of a character. For each occurrence the character gets a small mark in the belonging Fingerprint at the correlate passage in the document. The user can set the parts and chapters into the Fingerprint by clicking on Toggle seperators. To select a character he has to click on the name of the desired character in the list. For Deselection the user has to click again on the name in the list. He also has the ability to select multiple characters to get the Combination Fingerprint. The Combination Fingerprint is a Fingerprint of the selected characters. If the chosen characters occur together in a sentence, they’ll get a small mark like for a single character.
If the user clicks on one of the small mark, he will be forwarded to the Document. In the Document the citation will be highlighted with a specific color.
