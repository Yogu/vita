Overview

The Overview – a document is imported/chosen and the analysis and interaction can start. This view gives the user some general facts about the document, for example the most frequent occurred characters. Furthermore the Overview gives you links to the various analyzations. Thereby, the user has the opportunity to choose the right label in the list of interaction possibilities on the left, or he can select the corresponding picture. Besides the Overview, the tool ViTA offers 7 other views on the chosen document.

Graph Network

1 The list of offered interaction possibilities, including the Fingerprint, the Graph Network, the Word Cloud, the Plot View and the profile of a character/place

2 The general information about the document:¬¬ the name of the author, the publish year, the main characters, the number of chapters and the total count of words

3 The corresponding pictures to the list of offered interaction

The visualization Graph Network is a representation with nodes and edges of the connection between the characters in the document. Each node represents a character and each edge represents the connection between the two characters. The more important the character, the bigger the node. The more the characters occurs together in a sentence, the wider the edge between the nodes. Right at the start, the most important characters (with the maintained color) are already selected and shown in the visualization. The most common characters with the assigned color are shown in the Graph Network with the same color. The user can add more characters to his selection or deselect characters by clicking on the desired name in the list on the right. With the selection, the border of the name is getting fat. If the user clicks on an edge between two nodes, the Fingerprint between the two characters will be displayed above the Graph Network. With a click on the citation in the Fingerprint, the user is forwarded to the Document.
